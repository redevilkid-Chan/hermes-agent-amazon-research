# 9大AgentSkills 使用指南
## 触发方式 × 使用场景 × 实例

---

# Skill #1：Agent Reach
## 全网AI Agent情报站

**触发词**：Agent Reach / 研究AI Agent / AI工具情报

**核心功能**：搜索GitHub热门AI Agent开源项目，支持多AI并行交叉验证调研，发现最优方案

**使用场景**：
- "研究GitHub上前50的AI Agent项目"
- "帮我找有没有类似Manus的开源替代"
- "DeepSeek有哪些好用的Agent工具？"

**使用方式**：直接告诉我研究需求，我会调用Agent Reach执行

---

# Skill #2：YouTube Watcher
## 视频内容一键提取

**触发词**：YouTube Watcher / 油管 / 看视频 / 提取字幕

**核心功能**：抓取YouTube视频字幕，转为文字用于总结、问答、信息提取

**使用场景**：
- "帮我看这个视频说了什么：https://youtube.com/..."
- "这个教程视频的核心步骤是什么？"
- "提取这个访谈视频的关键观点"

**使用方式**：
```
python3 {baseDir}/scripts/get_transcript.py "YouTube链接"
```
⚠️ 需要安装 yt-dlp：`brew install yt-dlp`

# Skill #3：Summarize
## 万物皆可摘要

**触发词**：Summarize / 总结 / 摘要 / 提炼要点

**核心功能**：总结网页、PDF、图片、音频、YouTube视频为干净摘要

**使用场景**：
- "帮我总结这篇长文：https://..."
- "这个PDF的核心结论是什么？"
- "这个播客40分钟说了什么？"
- "这张图里的关键信息是什么？"

**使用方式**：直接发链接或文件路径给我

---

# Skill #4：Humanizer
## AI文风润色器

**触发词**：Humanizer / 润色 / 更像人写的 / 去AI味

**核心功能**：消除AI生成的套路感——过度使用的连接词、过于工整的段落结构、生硬的措辞

**使用场景**：
- AI写完文案后："帮我润色一下，更像人话"
- "这段太像ChatGPT写的了，帮我改改"
- 适合：朋友圈文案、微博、知乎回答、私信

**使用方式**：直接发文字给我润色

---

# Skill #5：Skill Vetter
## 安全门卫·安装前必扫

**触发词**：Skill Vetter / 扫描 / 安全检查 / 安装前审查

**核心功能**：安装新Skill前自动扫描——来源信誉、代码审计（检测可疑curl/eval/SSH访问）、风险评级

**使用场景**：
- "帮我扫描一下这个Skill安全吗？"
- 安装任何第三方Skill前先问一句
- 适合：刚发现的新Skill、来源不明确的Skill

**风险等级**：🟢低 → 🔴极高，给出建议

---

# Skill #6：Self-Improving Agent
## 你的专属"错题本"

**触发词**：Self-Improving Agent（自动生效，无需主动触发）

**核心功能**：自动记录执行错误和用户纠正，形成反馈闭环，下次自动调整

**使用场景**：
- 无需主动调用！平时使用中自动工作
- 你纠正我一次 → 我记住，下次不再犯
- 越用越懂你的习惯和偏好

**效果**：长期使用后，犯错率显著降低

# Skill #7：Tavily Search
## 精准搜索·直接给答案

**触发词**：Tavily / 搜索调研 / 深度研究 / 技术选型

**核心功能**：AI优化的搜索引擎，返回结构化高相关结果，不只是链接列表

**使用场景**：
- "帮我用Tavily研究一下RAG和Agent的区别"
- "搜索最近亚马逊VC账号的最新政策"
- "深度调研肌酸市场的最新竞品动态"
- 需要免费API Key（gmail邮箱注册送$10额度）

**优势**：比普通搜索更精准，不只是给你链接让你自己筛

# 使用建议

**日常工作流**：
- 遇到长文/视频 → Summarize / YouTube Watcher
- 写完文案 → Humanizer 润色
- 研究新领域 → Tavily Search + Agent Reach
- 发社交媒体 → 宝玉图文系列
- 做原型 → Frontend Design

**安全习惯**：
- 安装任何新Skill前 → Skill Vetter 扫描
- 长期使用 → Self-Improving Agent 自动优化

**一句话总结**：这9个Skills覆盖了「调研→消化→创作→发布→迭代」的全链路！
